description - nodeMemory Class
This class is the "brains" of the
AI class. It is used to manage nodes
that AI's will utilize.
See: AI.cs class.

Method 1

nodeMemory::addToMemory(%this,%position,%brick)

Notes: adds a node to the nodeMemory class of a specific AIPlayer utilizing this class

Method 2

nodeMemory::isInMemory(%this,%position)

Notes: checks to see if a position is actually a node and is in the nodeMemory's memory

Method 3

nodeMemory::clearMemory(%this)

Notes: clears all nodes from the nodeMemory

Method 4

nodeMemory::clearNodeByTime(%this)

Notes: Clears all nodes that the AIPlayer hasn't used in a while, to prevent potential memory hogging.
