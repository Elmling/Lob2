Class Description

author: Elm

description - Sounds Class
This class handles all sounds, music, ambience, etc.

classes: Sounds

Method 1

sounds::play2D(%this,%clientOrPlayer,%sound)

Notes: plays a sound 2D.

Method 2

sounds::play3D(%this,%posOrObj,%sound)

Notes: plays a sound 3D.

Method 3

sounds::register(%this,%name,%obj)

Notes: registers a sound into the sounds class

Method 4

sounds::getSound(%this,%sound)

Notes: Attempts to get an object from the sounds class containing data about the sound.

Method 5

sounds::display(%this,%sound)

Notes: writes to the console the current sounds registered.
