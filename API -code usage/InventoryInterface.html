description - InventoryInterface Class
This is the interface (centerprint)
used to work with and display inventory item

Notes: Displays the inventory to the client. Allows for cycling, notice: %index

InventoryInterface::hide(%this,%inventoryObject)

Notes: Removes the interface from the client's screen

Method 3

Notes: scrolls down the inventory interface's list

Method 4

InventoryInterface::scrollUp(%this,%inventoryObject)

Notes: scrolls up the inventory interface's list
