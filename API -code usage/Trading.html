Class Description

author: Elm

description - Trading Class
Handles trading items between two players

Method 1

trading::newTrade(%this,%client1,%client2)

Notes: attempts to create a new trade session between %client1 and %client2

Method 2

tradeSession::addItem(%this,%client,%item,%amount)

Notes: attempts to add an item into the tradeSession object

Method 3

tradeSession::getItem(%this,%item)

Notes: returns an item from the trade session argument should be a string

Method 4

tradeSession::removeItem(%this,%client,%item,%amount)

Notes: attempts to remove an item from the tradeSession object

Method 5

tradeSession::accept(%this,%client)

Notes: %client accepts the trade. once both clients accept, we can call the ::exchange

Method 6

tradeSession::resetAccepted(%this)

Notes: resets the accepted value between two clients.

Method 7

tradeSession::cancel(%this,%client)

Notes: will cancel the trade session between two users

Method 8

tradeSession::exchange(%this)

Notes: exchanges the current items in the session between the two users trading.

Method 9

tradeSession::endTrade(%this)

Notes: ends the trade session
